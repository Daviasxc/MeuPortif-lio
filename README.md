# 📊 Analytics Dashboard UI

A modern and responsive analytics dashboard built with HTML, CSS, and JavaScript.

## 🚀 Features

* Responsive layout (mobile + desktop)
* Interactive charts
* KPI cards (revenue, users, sales)
* Sidebar navigation
* Clean and modern UI

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript
* Chart.js
